## 📝 AI Content Generation Prompts

### Save These Prompts for Claude/ChatGPT

#### Prompt 1: Weekly Content Batch
```
You are a social media manager for Callyy, an AI voice agent platform for Indian businesses.

Product Details:
- AI voice agents that handle customer calls 24/7
- Pay-as-you-go pricing in ₹INR
- Supports Hindi, English, Tamil, Telugu, Bengali + more
- Customer Memory: AI remembers past conversations
- 5-minute setup, no coding required
- Target: Indian SMBs, startups, e-commerce

Generate a week's worth of social media content:

1. LinkedIn Post (Monday) - Industry insight with data
2. LinkedIn Post (Wednesday) - Product feature highlight
3. LinkedIn Post (Friday) - Founder story or customer success
4. Twitter Thread (Tuesday) - Educational content (5-7 tweets)
5. Twitter Posts (Daily) - 5 standalone tweets (tips, hot takes, engagement)
6. Threads Posts (2x) - Personal/behind-the-scenes content

Requirements:
- Use conversational Indian English
- Include relevant emojis
- Add call-to-actions
- Reference ₹INR for pricing
- Keep authentic founder voice
```

#### Prompt 2: Transform Blog to Social
```
Transform this blog post/idea into social media content for Callyy:

[PASTE YOUR CONTENT]

Create:
1. LinkedIn post (hook + story + CTA)
2. Twitter thread (7 tweets)
3. 5 standalone tweets
4. Threads post (personal angle)
5. LinkedIn carousel (8 slides outline)

Brand voice: Professional but friendly, India-focused, startup energy
```

#### Prompt 3: Daily Engagement Content
```
Generate 10 engagement-focused tweets for Callyy (AI voice platform):

Types needed:
- 2 polls about customer service/AI
- 2 hot takes on industry trends
- 2 questions to spark discussion
- 2 relatable observations
- 2 quick tips

Keep them conversational and under 280 characters.
```

## 🔧 Step-by-Step Automation Setup

### Step 1: Set Up Buffer (Free)
1. Go to buffer.com → Sign up free
2. Connect: LinkedIn, Twitter, Threads (Instagram for Threads)
3. Set posting schedule:
   - LinkedIn: Mon 9AM, Wed 12PM, Fri 6PM IST
   - Twitter: Daily 9AM, 1PM, 6PM IST
   - Threads: Daily 10AM, 7PM IST
4. Upload your content weekly

### Step 2: Set Up Typefully (Twitter-focused)
1. Go to typefully.com → Sign up
2. Connect Twitter account
3. Features to use:
   - Thread composer (easier than Twitter)
   - Auto-split long content
   - Schedule threads
   - Analytics

### Step 3: Set Up Notion Content Calendar
```
Database Properties:
- Title (text)
- Platform (select: LinkedIn/Twitter/Threads)
- Content Type (select: Post/Thread/Carousel/Video)
- Status (select: Idea/Draft/Ready/Scheduled/Published)
- Publish Date (date)
- Content (text)
- Visual (files)
- Performance (number)
```

### Step 4: Set Up Make.com/Zapier Automation

#### Automation 1: RSS to Social
```
Trigger: New blog post published
Action: 
  1. Send to Claude API → Generate social posts
  2. Create draft in Buffer
  3. Notify on Slack/Email
```

#### Automation 2: Weekly Content Reminder
```
Trigger: Every Sunday 10 AM
Action:
  1. Send email: "Time to create this week's content"
  2. Include content prompts
  3. Link to Notion calendar
```

#### Automation 3: Engagement Alerts
```
Trigger: New mention on Twitter/LinkedIn
Action:
  1. Send Slack notification
  2. Log in spreadsheet
  3. Create follow-up task
```

## 💡 Pro Tips for Automation

### Do's ✅
- Always review AI-generated content before posting
- Keep 20% of content spontaneous/real-time
- Respond to comments personally (don't automate)
- A/B test different content types
- Track what works and double down

### Don'ts ❌
- Don't auto-DM new followers (looks spammy)
- Don't use engagement pods (platforms detect this)
- Don't post identical content across platforms
- Don't over-automate (lose authenticity)
- Don't ignore analytics
