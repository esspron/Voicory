## Implementation Notes

### For Next.js Website (`website-nextjs/`)

Create pages at:
- `src/app/terms/page.tsx`
- `src/app/privacy/page.tsx`
- `src/app/refund/page.tsx`
- `src/app/cookies/page.tsx`

### For React Dashboard (`frontend/`)

Add routes for these pages in the public area (accessible without login).

### Footer Links

Update the footer on both website and dashboard to include:
```html
<footer>
    <a href="/terms">Terms of Service</a>
    <a href="/privacy">Privacy Policy</a>
    <a href="/refund">Refund Policy</a>
</footer>
```

### Paddle Checkout Display

Paddle will automatically show links to your Terms and Privacy pages in the checkout. Make sure these URLs are configured in your Paddle dashboard under Settings → Checkout.
