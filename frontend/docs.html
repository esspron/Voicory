What is Voice AI?

Voice AI enables you to create intelligent conversational agents that can understand natural language, respond contextually, and perform complex tasks through voice interactions. Our platform provides:

  • Real-time Speech Processing - Process and understand speech with ultra-low latency
  • Natural Conversations - Engage users with human-like dialogue flows
  • Function Calling - Execute custom functions based on conversation context
  • Webhook Integration - Connect to your existing systems seamlessly

Getting Started

Follow these steps to create your first voice AI assistant:

1. Install the SDK

Install our SDK using your preferred package manager:

npm install @voice-ai/sdk
# or
yarn add @voice-ai/sdk
# or
pnpm add @voice-ai/sdk

2. Initialize the Client

Set up the SDK with your API credentials:

import { VoiceAI } from '@voice-ai/sdk';

const client = new VoiceAI({
  apiKey: process.env.VOICE_AI_API_KEY,
});

// Create an assistant
const assistant = await client.assistants.create({
  name: 'My First Assistant',
  voice: 'en-US-Neural2-F',
  model: 'gpt-4',
  systemPrompt: 'You are a helpful assistant.'
});

3. Make a Call

Initiate a call with your newly created assistant:

const call = await client.calls.create({
  assistantId: assistant.id,
  phoneNumber: '+1234567890',
  from: '+0987654321'
});

console.log('Call initiated:', call.id);

Quick Examples

Example Customer Support Bot

Create an AI assistant that handles customer support queries, checks order status, and escalates to human agents when needed.

const supportAssistant = await client.assistants.create({
  name: 'Support Bot',
  voice: 'en-US-Neural2-C',
  model: 'gpt-4',
  systemPrompt: `You are a customer support assistant. 
    Help customers with order inquiries, product questions, 
    and general support. Be friendly and professional.`,
  tools: [
    {
      type: 'function',
      function: {
        name: 'check_order_status',
        description: 'Check the status of a customer order',
        parameters: {
          type: 'object',
          properties: {
            orderId: { type: 'string' }
          }
        }
      }
    }
  ]
});
